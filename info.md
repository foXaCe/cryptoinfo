# Cryptoinfo

Track cryptocurrency prices and Bitcoin mining statistics in Home Assistant.

## Features

- **Price Tracking**: Monitor any cryptocurrency via CoinGecko API
- **Portfolio Calculation**: Multiply by your holdings to see total value
- **Bitcoin Network Stats**: Track hashrate, difficulty, and block height
- **Bitcoin Mempool Stats**: Monitor mempool size and fee estimates
- **CKPool Mining Stats**: Track your solo mining performance

## Installation

1. Add this repository to HACS
2. Install "Cryptoinfo"
3. Restart Home Assistant
4. Go to Settings → Integrations → Add Integration → Cryptoinfo

## Configuration

Configure via the UI:
- Select sensor type (Price, Network, Mempool, or Mining)
- Search and select cryptocurrencies
- Set update frequency and multipliers
